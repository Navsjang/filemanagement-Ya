File Management

1. Why File Management Matters

File management is the process of organizing, naming, storing, and maintaining digital files so you can easily find, use, and protect them.

Why It Matters

  • ✅Saves Time
    You can quickly locate files instead of searching through cluttered folders.
  • ✅Prevents Lost Work
    Correct storage and backups protect your data from deletion or corruption.
  • ✅Improves Productivity
    A clean, logical structure helps you work faster and collaborate better.
  • ✅Avoids Duplication
    Organized files reduce confusion and mistakes caused by multiple versions.

Example

If your project is well-organized (e.g., Documents > School > ICT > File Management), you can instantly find your homework.

Without a structure, files may be scattered everywhere, causing delays.

2. Navigate File Manager UI and Interpret Properties/Metadata

Every operating system (Windows, macOS, Linux) has a file manager (Windows Explorer, Finder, etc.).

Key Parts of a File Manager UI

  • ✅Navigation Pane
    sidebar with quick access to folders (Documents, Downloads, etc.)
  • ✅Search Bar
    find files by name or type

File Properties / Metadata

Metadata is data about a file, including:

PropertyWhat It Means
NameFile's title
TypeFile format (PDF, DOCX, JPG...)
SizeHow much storage it uses
Date ModifiedLast update time
Created DateWhen the file was made
Location/PathWhere the file is stored
Author(Some file types) who edited/created it

3. Apply Naming Conventions and Versioning

  • ✅Use descriptive filenames
    include keywords about the content
  • ✅Avoid spaces and special characters
    use underscores or camel case
  • ✅Add version numbers
    indicate updates or revisions

Example:

  • FileManagement_Presentation.pptx
  • FileManagementPresentation_v3.pptx
  • Filemanagementpresentation_final.pptx
  • FileManagementPresentation_FinalVersion.pptx
